# HTTP API contract

The machine-readable contract for chatlab's HTTP surface is **[OpenAPI 3.1 YAML](./openapi.yaml)**.

Use the **OpenAPI reference** entry in the top navigation (Redoc UI) for an interactive, searchable view of every path, schema, and response code.

There are no separate `http.md` / `webhooks.md` prose files in this repository; narrative documentation lives in capability specs under [`specs/capabilities/` on GitHub](https://github.com/jvrmaia/chatlab/tree/main/docs/specs/capabilities) and in [`recipes.md`](/recipes).
