# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1**.

The full text is published at:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

By participating in this project — as a contributor, maintainer, reviewer, or community member — you agree to uphold the standards described there.

## Reporting

To report behavior that violates the Code of Conduct, contact the maintainers privately at the email listed in [`SECURITY.md`](./SECURITY.md). Reports are handled confidentially. Maintainers commit to acknowledging reports within 7 days.

## Enforcement

Maintainers will follow the Contributor Covenant Enforcement Guidelines (Correction → Warning → Temporary Ban → Permanent Ban) when responding to violations.
